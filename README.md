# Trading Intelligence Platform

An end-to-end stock market analysis platform that combines Machine Learning, Technical Analysis, News Sentiment Analysis, and Strategy Backtesting to help users make data-driven trading decisions.

The platform analyzes historical stock data, generates technical indicators, evaluates market sentiment from recent news, predicts market direction using Machine Learning, and provides Buy, Hold, or Sell recommendations through an interactive dashboard.

---

## Features

- Historical stock market data analysis
- Technical indicator generation
- Real-time news sentiment analysis
- Machine Learning-based market prediction
- Buy / Hold / Sell recommendations
- Strategy backtesting
- Risk and performance evaluation
- Portfolio growth comparison
- Interactive visualizations and dashboards

## Installation

```bash

cd trading-intelligence-platform
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```
