# 📈 HellStock – Stock Analysis & Trading Simulator

HellStock is an interactive stock analysis and trading simulation platform built using Python and Streamlit. It enables users to analyze real-time stock data, visualize trends, and simulate trading strategies for better investment decision-making.

---

## 🚀 Features

- 📊 Real-time stock data fetching using yFinance API  
- 📈 Interactive data visualization using Plotly  
- ⚙️ Configurable trading parameters (buy/sell thresholds, risk-free rate)  
- 🔍 Data analysis using Pandas  
- 🧪 Train-test split for backtesting strategies  
- 💡 User-friendly interface built with Streamlit  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Pandas  
- NumPy  
- Plotly  
- yFinance API  

---

## 📂 Project Workflow

1. Fetch real-time stock data using yFinance  
2. Perform data preprocessing and analysis using Pandas  
3. Visualize stock trends using Plotly  
4. Apply trading strategy logic with user-defined parameters  
5. Evaluate performance using backtesting (train-test split)
